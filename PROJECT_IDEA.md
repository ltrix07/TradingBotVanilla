# Project Idea: Polymarket Paper Trading Bot

**Problem**: Эмоциональный трейдинг на 5-минутках и высокие комиссии.
Ручной мониторинг 24/7 невозможен; рыночные ордера съедают прибыль через проскальзывание.

**Solution**: Автономный Python-бот (Fast MACD 3-15-3) с локальной симуляцией торговли (Paper Trading).
Бот использует REST API для получения данных и имитирует торговлю через `state.json` без финансового риска.

**Architecture**:
- **Discovery Layer** — Автоматический поиск активных 5-минутных рынков через Polymarket Gamma API.
- **Data Layer** — REST API (Binance для цены BTC, Polymarket CLOB для стаканов)
- **Strategy Layer** — Расчёт индикаторов MACD (3, 15, 3) через `pandas_ta`
- **Risk Engine** — Контроль виртуального баланса, SL/TP и лимитов просадки
- **Execution Layer (Mock)** — Локальный брокер, обновляющий `state.json` вместо блокчейна

**Stack**: Python 3.10+, Pandas, Pandas_TA, HTTPX, PyYAML

**Target audience**: Разработчики и алготрейдеры, желающие автоматизировать стратегии на Polymarket.

## Риски

| Риск | Вероятность | Митигация |
|------|-------------|-----------|
| Ошибка API / Таймаут | Высокая | Логирование, пропуск итерации без сделок |
| Проскальзывание | Средняя | Учёт Bid/Ask из стакана в симуляторе |
| Переобучение стратегии | Средняя | Обязательный этап инкубации (Paper Trading) |
